# Mini-Apps

A collection of small JavaScript projects built to practice core frontend development skills.

This repository currently includes:

- **Bug Tracker** – add, manage, filter, and resolve bugs
- **Fighter Arena** – create fighters, track health, and simulate simple battle logic

## Purpose

The goal of this repository is to improve practical skills in:

- JavaScript fundamentals
- DOM manipulation
- Event handling
- Array methods
- State management
- Dynamic rendering

## Tech Stack

- JavaScript
- HTML
- CSS

## Why this project?

Instead of building only one large app, this repository contains smaller focused applications that help strengthen problem-solving and build a solid development foundation.
